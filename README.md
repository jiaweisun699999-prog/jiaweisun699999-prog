### ⚙️ 技术栈与核心架构 (Tech Stack)

* **AI 全栈工程 (LLM & RAG):** 熟练掌握开源模型（Qwen, Llama）本地部署与 LoRA 微调。擅长构建低成本、高并发的流式对话架构（FastAPI + SSE + 外部 API 如 DeepSeek）。
* **视觉与自动化 (Computer Vision):** 深度定制 ComfyUI 与 Stable Diffusion 工作流，**专注于电商摄影的服装瑕疵自动化修复**，实现高杠杆的图像后处理替代。
* **数据与量化 (Quant & Data):** 构建基于大数据的金融研报 RAG 检索系统，开发量化终端进行持仓分析与宏观周期（半导体、基础金属）追踪。
* **具身智能 (Embodied AI):** 探索 LLM 在复杂环境中的智能体表现（如 Minecraft 服务端 AI Agent 行为脚本化）。

### 🚀 核心开源项目 (Featured Projects)
